# ferengi
marketing wabsite

```
// Install the dependencies
npm install

// Build the static site. Output will be in dist/
npm run build

// Build the static site and watch for any changes
npm run build:watch

// Run the tests. Everything must lint and build before it can get
// merged onto master.
npm run test

// Deploy to production
grunt deploy
```
